# Grill me — abd-domain-specification

**Mechanics:** [`common/reference/grill-me-with-practice-skill.md`](../../../../common/reference/grill-me-with-practice-skill.md) — one question at a time; generate-to-learn when enough is shared.

- Which properties are generic primitives when the domain actually constrains them?
- When two concepts reference each other, which one owns the relationship — what happens when the owner changes?
- Which business rules span multiple classes — where does enforcement live?
- For multi-step operations, are there hidden collaborators the sequence depends on?
- Which concepts have identity vs value semantics — what breaks if that's wrong?
