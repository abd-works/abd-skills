# Grill me — abd-bounded-context-map

**Mechanics:** [`common/reference/grill-me-with-practice-skill.md`](../../../../../common/reference/grill-me-with-practice-skill.md) — one question at a time; generate-to-learn when enough is shared.

- Which contexts look independent but share data, vocabulary, or lifecycle?
- Where does one team's responsibility end and another's begin — does everyone agree?
- Which boundary crossings corrupt meaning because the same word means different things?
- Is there a bounded context nobody has named yet?
- For each dependency, which side sets the rules — what happens when downstream disagrees?
