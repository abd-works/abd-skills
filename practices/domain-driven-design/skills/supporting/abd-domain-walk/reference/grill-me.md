# Grill me — abd-domain-walk

**Mechanics:** [`common/reference/grill-me-with-practice-skill.md`](../../../../../common/reference/grill-me-with-practice-skill.md) — one question at a time; generate-to-learn when enough is shared.

- Which scenarios have you walked only as happy paths — what happens on failure or missing data?
- Which pseudocode line required inventing a method or class not in the domain model?
- What happens when two aggregates must coordinate and one rejects the request?
- Are scenario values realistic enough to expose edge cases?
- Which business scenario did the domain expert mention that you haven't walked yet?
