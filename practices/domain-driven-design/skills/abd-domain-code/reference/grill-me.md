# Grill me — abd-domain-code

**Mechanics:** [`common/reference/grill-me-with-practice-skill.md`](../../../../common/reference/grill-me-with-practice-skill.md) — one question at a time; generate-to-learn when enough is shared.

- Which business rules feel obvious but haven't been stated — what breaks if code doesn't enforce them?
- When two domain objects could own the same operation, which one actually does?
- Which state transitions are legal vs rejected — have we mapped them all?
- Where is the boundary between pure domain logic and infrastructure — what's leaking across?
- What happens at the limits — zero items, max quantities, duplicates — where rules interact?
