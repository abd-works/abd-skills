# Source References

References are grouped by KA — one `### KA References` section per KA, after all its terms. Every term must be covered by at least one reference entry. References point to an exact file and location on disk.

## Format

```
**Ref — {{short title}}**
Source: {{relative_path_to_source_file}}
Locator: {{lines / chapter / page}}
Extract: {{whole | partial}}
```

See `../templates/module-file-template.md` for the canonical file shape.
