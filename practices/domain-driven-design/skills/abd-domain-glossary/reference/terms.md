# Domain Terms

A **domain term** is a named concept from the domain. For each term, capture behavioral bullets grounded in source material.

## Deciding where a term belongs

- **Keep under a KA** — term has clear meaning within this module.
- **Move to boundary** — this module depends on it but doesn't own it → `# Boundary Domain` with `*(owned by: Module)*`.
- **Move to another module** — term belongs elsewhere entirely.

## Boundary terms

A concept this module depends on but does not own. Appears under `# Boundary Domain` as `### term *(owned by: Module)*`.
