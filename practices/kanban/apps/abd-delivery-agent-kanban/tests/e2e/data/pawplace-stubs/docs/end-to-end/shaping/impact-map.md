# Impact map — PawPlace (stub)

## Goal

Drive foot traffic to stores by showing real-time product stock online.

## Actors

| Actor | Impacts |
| --- | --- |
| Shopper | Finds products quickly; trusts stock shown before visiting |
| Store employee | Updates stock so online view matches shelf |

## Deliverables

| Option | Supports impact |
| --- | --- |
| Keyword search + product detail | Shopper finds the right product |
| Per-store stock display | Shopper walks in with confidence |
