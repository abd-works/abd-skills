# Domain Language — Increment 1 (stub)

Scope: Find products and check store stock.

**Terms:**
- **Product Catalog**
  - **product catalog** — searchable collection of pet supplies
  - **product** — item for purchase with SKU and price
  - **category** — grouping for browse/filter
  - **stock availability** — real-time quantity at a store
- **Store**
  - **store** — physical location with address
  - **store locator** — discovery feature (map/list)

Boundary: *product detail page*, *search results*.
