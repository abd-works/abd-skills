# Grill me — drawio-story-sync

Only when the invocation includes **"grill me"**.

- Which diagram is the source of truth — outline, acceptance-criteria, or thin-slicing — and which command will you run?
- What report diff will `sync` produce — and which persona or rename changes apply to the graph?
- Which companion diagrams will be refreshed — and why?
- What validation will you run on `story-graph.json` after the write?
- What sidecar files (`*-extracted.json`, `*-update-report.json`) will be removed after a successful sync?
