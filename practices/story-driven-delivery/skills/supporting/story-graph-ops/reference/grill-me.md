# Grill me — story-graph-ops

Only when the invocation includes **"grill me"**.

- What CLI commands will you run to validate the graph — at minimum `read`?
- How will you capture and pass `--expect-sha` if another writer might touch the file?
- Which parser script matches the markdown source — story map, AC, or thin-slicing?
- What anti-pattern are you avoiding — hand-rolled JSON, foreign loaders only, or skipped PYTHONPATH?
- What will you report as proof this skill was followed — not only "wrote JSON"?
