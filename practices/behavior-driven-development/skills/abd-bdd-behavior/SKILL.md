# abd-bdd-behavior

## Purpose

Translate domain practice artifacts into a **plain-English behavior hierarchy** — the first artifact in a BDD workflow. The scaffold names every domain concept, state, and behavior in natural language before any code is written, anchored to the sub-epics in the story map.
