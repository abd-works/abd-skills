# Input traps — abd-bdd-specification

Pre-flight only — not grill questions.

- **Scaffold completeness** — Is the scaffold fully approved, or are there describe blocks with open questions?
- **Framework choice** — Is the target framework confirmed? Jest and Mamba have different nesting syntax.
- **Hierarchy fidelity** — Does every scaffold level have a code equivalent? Collapsed levels are hard to recover in the test phase.
- **File target** — Does a test file already exist? If yes, add to it; if no, confirm the file name before writing.
