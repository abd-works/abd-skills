# conf/

- **`abd-config.json`** — workspace routing at install time. Set **`active_skill_workspace`** to the project root you work against (or **`.`** when the skill directory is the workspace). Same shape as **`skills/abd-skill-builder/conf/abd-config.json`**.

- **`build-strategy.json`** — build intent / strategizer payload for tools that consume it. Fill **`skill_purpose`** when you wire **`agentic-skill-builder`**.
