# `library/required/` — fill these for your skill

These markdown files are **templates** created by **`scaffold_skill.py`**. Replace placeholders (`{{skill_name}}`, …) and extend the content so they describe **this** skill’s purpose, outline, role, and principles.

Frozen shared norms live under **`library/base/`** (copied from **abd-skill-builder**, no routine edits). Additional cross-cutting docs can be added at **`library/*.md`** and listed in **`skill-config.json` → `library_files`**.

See **`content/parts/library/required/README.md`** in the **abd-skill-builder** repo for the full convention.
