Agents live under agents/ as one folder per agent. The catalogue uses the first existing entry file in this order: AGENT.md, AGENTS.md, then SKILL.md. Each card opens an agent detail page with the same structure as skills: full description (Purpose-first, same heuristics as skills), ASCII package diagram, and contents with folder summaries plus nested file rows. Repository links open in a new tab. Card summaries are auto-derived for quick scanning; use the detail page for a complete picture of what the agent is for.
