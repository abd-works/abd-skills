---
Template reference only — do not copy the following into generated project files:
- Given = state; first action is When; Then = domain-visible outcomes.
- Domain language, not UI widget names; {Concept} ↔ example table bidirectionally; put domain words beside each brace in steps (e.g. the User {User}).
- In paired .md files, use *italics* for domain-significant terms (plain .txt has no markdown).
- Happy path + edges + errors implied by the story, notes, or AC (if any); avoid outlines for single-row or unrelated setups.
- For story-graph workflows, persist scenarios in story-graph.json, not ad-hoc spec files.
