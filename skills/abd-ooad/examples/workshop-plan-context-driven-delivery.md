## Session 2: Hands-on skill-building session

**Time:** 10:30–12:00 (1.5 hrs)  
**Header:** Orange

1. **Copilot 101** (pink)
2. **What is a skill?** (pink)
   - **Structure of a skill** (pink)
   - **The `SKILL.MD` framework (flow diagram)**  
     **Inputs → `SKILL.MD` → perform**

     | Component | Role |
     |-----------|------|
     | `input` | Context that **must** be used by the model |
     | `context` | Relevant background the model **must** know |
     | `Persona` | Persona, role, expertise the model **must** act as |
     | `Instruction` | Standards and rules; sequence of steps the model **must** follow |
     | `Output` | Output structure / format / schema the **agent** should follow |
     | `README.MD` | Generated by the model, read by humans |

     - Central box: **SKILL.MD**
     - Outcome (green): **Perform** — artifacts produced by the model
3. **Building a skill** (pink)
   - Example task: *adding acceptance criteria to stories*
