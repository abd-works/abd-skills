# Shared DrawIO Python modules

**Canonical sources** live here. Edit these files only—not the copies under `skills/*/scripts/`.

Those copies are **generated** with a timestamped header:

```bash
python scripts/sync_drawio_vendor.py
```

Run from the **agilebydesign-skills** repository root. The root `build.py` runs this sync; `skills/abd-story-synthesizer/scripts/build.py` also invokes the sync when building the synthesizer skill.

See also **[../README.md](../README.md)** for the full `src/` layout and check workflow.
